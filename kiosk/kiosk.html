Background Info:

Chromium browser is automatically startet by file:

  • /home/pi/.config/autostart/Chromium-Starter.desktop

with local website:

  • /home/pi/kiosk/kiosk.html

What you have to to:

  1. stop fullscreen modus with F11 and stop Chromium
  2. open Directory /home/pi/kiosk with fileexplorer
  3. rename kiosk.html to kiosk.original.html
  4. copy kiosk_1_Page.html to kiosk.html, if you need ONE kioskpages
    copy kiosk_N_Pages.html to kiosk.html, if you need TWO or MORE kioskpages
  5. open new kiosk.html in texteditor and change variable:
    1. header, or header1/header2/...
    2. url, or url1/url2/...
    3. if you need page changes in minutes instead hours, fix "var currentTime = new Date().getHours();" to "var currentTime = new Date().getMinutes();"
    to your needs.
  6. start Chromium with double click on file:
    • /home/pi/.config/autostart/Chromium-Starter.desktop
